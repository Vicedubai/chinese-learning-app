## 🐛 TROUBLESHOOTING

### Edit not working?
- Click ✏️ button (not the book title)
- Type new name
- Press Enter

### Drag not working?
- Click and hold on book (not on buttons)
- Drag to new position
- Release to drop

### Changes not saving?
- Refresh page
- Check if data persists
- Report if not
