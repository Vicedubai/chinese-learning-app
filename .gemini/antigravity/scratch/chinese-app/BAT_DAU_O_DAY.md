## 🎯 DEPLOY NHANH (15 PHÚT)

```bash
# 1. Push lên GitHub (5 phút)
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/chinese-learning-app.git
git push -u origin main

# 2. Deploy Frontend (5 phút)
# Vào: https://vercel.com
# New Project → Import repo → Deploy

# 3. Deploy Backend (5 phút)
# Vào: https://railway.app
# New Project → Deploy from GitHub → Generate Domain

# 4. Kết nối (2 phút)
# Sửa js/core.js → Push lại
```

**Chi tiết**: Xem `DEPLOY_NHANH.md`

## 🆘 CẦN GIÚP?

### Lỗi thường gặp:
- **Git not found**: Cài Git và restart terminal
- **Permission denied**: Dùng Personal Access Token
- **API not working**: Kiểm tra URL trong js/core.js
- **Build failed**: Xem logs trong dashboard

### Tài liệu:
- Git: https://git-scm.com/doc
- Vercel: https://vercel.com/docs
- Railway: https://docs.railway.app
