# ⚡ SETUP YOUTUBE API - NHANH 5 PHÚT

## 🎯 MỤC TIÊU
Lấy YouTube API Key để tìm kiếm video trong app

---

## 📝 5 BƯỚC ĐƠN GIẢN

### 1️⃣ Vào Google Cloud Console
```
https://console.cloud.google.com
```
→ Đăng nhập Google

---

### 2️⃣ Tạo Project
- Click **"Select a project"** → **"NEW PROJECT"**
- Tên: `Chinese Learning App`
- Click **"CREATE"**

---

### 3️⃣ Enable YouTube API
- Menu: **"APIs & Services"** → **"Library"**
- Tìm: `YouTube Data API v3`
- Click **"ENABLE"**
Chinese Learning App
---

### 4️⃣ Tạo API Key
- Menu: **"Credentials"**
- Click **"+ CREATE CREDENTIALS"** → **"API key"**
- Click **"COPY"** để copy API Key

---

### 5️⃣ Paste vào App
- Mở app → Trang **🎧 Nghe chép**
- Nhập từ khóa tìm kiếm
- Paste API Key vào ô
- Click **"💾 Lưu"**

---
