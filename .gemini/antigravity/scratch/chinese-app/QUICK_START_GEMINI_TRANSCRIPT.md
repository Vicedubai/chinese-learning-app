# ⚡ QUICK START - GEMINI TRANSCRIPT HELPER

## 🎯 WHAT'S NEW

Nút mới: **✏️ Tạo script thủ công**

Chức năng:
1. Nhập YouTube link
2. Nhấn nút "Tạo script thủ công"
3. Mở Gemini AI (tự động)
4. Copy transcript từ Gemini
5. Dán vào app
6. Bắt đầu luyện nghe

---

## 🚀 HOW TO USE (3 STEPS)

### Step 1: Nhập YouTube Link
```
🎧 Nghe Chép → 🔗 Link YouTube
https://www.youtube.com/watch?v=...
```

### Step 2: Nhấn "Tạo Script Thủ Công"
```
✏️ Tạo script thủ công
→ Modal hiển thị
→ Nhấn 🚀 "Mở Gemini AI"
```

### Step 3: Copy & Paste
```
Gemini tạo transcript
→ Copy kết quả
→ Dán vào ô Transcript
→ Nhấn ✅ "Lưu & Đóng"
```

---

## 📝 TRANSCRIPT FORMAT

```
0:00 - 0:05
Xin chào, đây là video tiếng Trung.

0:05 - 0:10
Hôm nay chúng ta sẽ học từ vựng mới.

0:10 - 0:15
Bạn đã sẵn sàng chưa?
```
