# ✏️ GEMINI TRANSCRIPT HELPER - UPDATED

## 🔄 CHANGES

### Vấn Đề
- Gemini không lưu được conversation khi mở từ URL
- Cần phải nhập API Key để lưu

### Giải Pháp
- Sử dụng tính năng **AI Settings** đã có sẵn
- Nhập Gemini API Key vào AI Settings
- Copy prompt thủ công
- Paste vào Gemini
- Gemini sẽ lưu conversation

## 🚀 CÁCH SỬ DỤNG (Updated)

### Step 1: Cài Đặt Gemini API Key
```
1. Nhấn nút 🔑 "Gemini API"
2. Nhập Gemini API Key
3. Nhấn ✅ "Lưu cài đặt"
```

### Step 2: Nhập YouTube Link
```
🎧 Nghe Chép → 🔗 Link YouTube
https://www.youtube.com/watch?v=...
```

### Step 3: Tạo Script Thủ Công
```
✏️ Tạo script thủ công
→ Modal hiển thị
→ Nhấn 🚀 "Mở Gemini AI"
```

### Step 4: Copy Prompt
```
Prompt sẽ được copy tự động
Gemini mở trong tab mới
Paste prompt vào Gemini (Ctrl+V)
```

### Step 5: Gemini Tạo Transcript
```
Gemini sẽ tạo transcript
Conversation được lưu trên Gemini
```

### Step 6: Copy & Paste
```
Copy transcript từ Gemini
Quay lại tab app
Dán vào ô Transcript
Nhấn ✅ "Lưu & Đóng"
```

## 🔑 GEMINI API KEY

### Cách Lấy API Key
1. Vào [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Nhấn "Create API Key"
3. Chọn "Create new API key in new project"
4. Copy API Key
5. Dán vào ô "🔑 Gemini API Key" trong AI Settings

### Lợi Ích
- ✅ Lưu conversation trên Gemini
- ✅ Sử dụng offline mode
- ✅ Tăng giới hạn requests
- ✅ Truy cập lịch sử

## 📝 TRANSCRIPT FORMAT

### Định Dạng Chuẩn
```
[Thời gian bắt đầu] - [Thời gian kết thúc]
[Nội dung transcript]

[Thời gian bắt đầu] - [Thời gian kết thúc]
[Nội dung transcript]
```

### Ví Dụ
```
0:00 - 0:05
Xin chào, đây là video tiếng Trung.

0:05 - 0:10
Hôm nay chúng ta sẽ học từ vựng mới.

0:10 - 0:15
Bạn đã sẵn sàng chưa?
```

## ⚠️ TROUBLESHOOTING

### Problem 1: Prompt không copy
**Solution**: Nhấn nút "📋 Copy Prompt Thủ Công"

### Problem 2: Gemini không mở
**Solution**: Kiểm tra popup blocker, cho phép popup

### Problem 3: Transcript không lưu
**Solution**: Kiểm tra ô Transcript, dán lại, nhấn "Lưu & Đóng"

### Problem 4: Gemini không lưu conversation
**Solution**: Cài đặt Gemini API Key trong AI Settings

### Problem 5: API Key không hoạt động
**Solution**: 
1. Kiểm tra API Key có đúng không
2. Vào [Google AI Studio](https://aistudio.google.com/app/apikey)
3. Tạo API Key mới
4. Lưu lại
