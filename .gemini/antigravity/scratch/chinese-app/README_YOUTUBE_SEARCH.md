# 🎬 TÌM KIẾM YOUTUBE - HƯỚNG DẪN SỬ DỤNG

## ✨ TÍNH NĂNG

Tìm kiếm video YouTube để luyện nghe chép tiếng Trung.

---

## 🚀 CÁCH SỬ DỤNG

### Bước 1: Vào trang Nghe chép
Click **🎧 Nghe chép** ở sidebar

### Bước 2: Nhập từ khóa
```
Ví dụ:
- "học tiếng trung"
- "HSK 1"
- "luyện nghe"
- "chinese conversation"
```

### Bước 3: Xem kết quả
- Hiển thị 12 video
- Thumbnail + Thông tin đầy đủ
- Click thumbnail → Xem video

### Bước 4: Chọn video
- Click **"✅ Chọn video"**
- Video tự động load vào dictation
- Dán transcript hoặc dùng AI
- Bắt đầu luyện!

---

## 🔑 API KEY

### Mặc định:
- ✅ **Không cần setup** - Hoạt động ngay
- ✅ **Miễn phí** - Dùng API Key của app
- ⚠️ **Giới hạn** - 100 lần tìm kiếm/ngày (chia sẻ)

### Dùng API Key riêng (Tùy chọn):
1. Click nút **"🔑 API Key"** sau khi tìm kiếm
2. Nhập API Key của bạn
3. Click **"💾 Lưu"**

**Lợi ích:**
- ✅ Không giới hạn (10,000 requests riêng)
- ✅ Tốc độ nhanh hơn
- ✅ Không ảnh hưởng users khác

**Cách lấy API Key:** Xem `YOUTUBE_API_SETUP_NHANH.md`

---

## 💡 MẸO

### Tìm kiếm hiệu quả:
- Dùng từ khóa tiếng Anh: "chinese learning"
- Thêm level: "HSK 1", "beginner"
- Thêm skill: "listening", "conversation"

### Chọn video phù hợp:
- Xem mô tả trước khi chọn
- Chọn video có phụ đề
- Chọn độ dài vừa phải (5-15 phút)

---

## 🐛 TROUBLESHOOTING

### "Quota exceeded"
**Nguyên nhân:** Đã vượt 100 lần tìm kiếm/ngày

**Giải pháp:**
1. Đợi đến ngày mai
2. Hoặc dùng API Key riêng

---

### "API key not valid"
**Nguyên nhân:** API Key bị lỗi

**Giải pháp:**
1. Liên hệ admin
2. Hoặc dùng API Key riêng

---

### Video không load
**Nguyên nhân:** Video bị xóa/private

**Giải pháp:**
1. Chọn video khác
2. Hoặc tìm kiếm lại
